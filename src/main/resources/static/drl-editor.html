使用提示：

              
    
                
  • 包名 (Package): 定义规则所属的包
    • 
                
  • Unit 名称: 单元名称，用于规则组织
    • 
                
  • 导入 (Import): 导入需要的Java类
    • 
                
  • 全局变量 (Global): 定义全局对象，如logger
    • 
                
  • Salience: 规则优先级，数值越大越先执行
    • 
                
  • No-Loop: 防止规则循环触发
    • 
                
  • When: 规则条件（LHS - Left Hand Side）
    • 
                
  • Then: 规则动作（RHS - Right Hand Side）
    •
